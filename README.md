# navigation

一款美观实用的手机导航网页，支持网页搜索、常用网站的添加与删除功能。使用媒体查询完美支持移动端与 PC 端，自动获取常用网页的 favicon 图标或首字母，并且可以通过键入常用网页首字母实现快捷访问。
